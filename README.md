## 启动

克隆项目(dev分支即可)

```
git clone --depth 1 -b dev https://github.com/xu-ux/mall-mogujie.git
```

安装

```
npm install
```

开发环境，预览

```shell
npm run dev
```

打包，部署生产

```shell
npm run build:prod
```

❗ test环境使用的是mock数据 `npm run test`
